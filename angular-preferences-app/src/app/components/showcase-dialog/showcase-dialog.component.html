color_lens Color Vision Simulation

This application includes an accessibility tool to simulate how users with different types of color vision deficiency perceive the interface.

How to activate the filter:

  1. Locate the floating Quick Settings buttons in the bottom-right corner of the screen.
  2. Click the eye icon visibility to open the simulation menu.
  3. Select a filter (e.g., Protanopia, Tritanopia) to apply it across the page.
  4. To disable the filter, select Default from the menu.

This helps ensure the UI is accessible and has sufficient color contrast for all users.
